# Tutorial de Git + GitHub

Primera vez

1- Creo la carpeta contenedora

2- Ingreso a la carpeta por consola 

3- Clono el repositorio
		git clone miRepo

4- Hago los cambios

5- Agrego cambios
		git add .

6- Comitear cambios
		git commit -m "Mensaje"

7- Subir al repositorio web
		git push
